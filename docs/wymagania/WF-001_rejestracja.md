\# WF-001 Rejestracja



\*\*Aktor:\*\* Niezalogowany użytkownik  

\*\*Cel:\*\* Utworzenie konta w systemie MicroBlog  

\*\*Priorytet:\*\* Wysoki



\## Opis

Użytkownik może utworzyć konto podając dane rejestracyjne.



\## Kryteria akceptacji

\- System umożliwia rejestrację nowego użytkownika.

\- Po poprawnej rejestracji użytkownik może się zalogować.

\- System odrzuca rejestrację, jeśli dane są niepoprawne lub konto już istnieje.



\## Scenariusz główny

1\. Użytkownik otwiera formularz rejestracji.

2\. Wprowadza dane (np. login/email i hasło).

3\. Zatwierdza formularz.

4\. System tworzy konto i potwierdza rejestrację.



\## Scenariusze alternatywne

A1. Dane niepoprawne → system pokazuje błąd i nie zakłada konta.  

A2. Konto już istnieje → system informuje i nie zakłada konta.
